# 🤖 AI Classification Systems (Chapter 2)

This repository showcases my expertise in **Classification Tasks** and model evaluation techniques, developed during my specialization at AI Academy.

## 📂 Included Projects:

### 1. Customer Satisfaction Predictor
A business-focused model designed to classify e-commerce customers based on their experience.
- **Algorithms:** Logistic Regression.
- **Techniques:** SMOTE for imbalanced data, Pipelines, and GridSearchCV for optimization.
- **Outcome:** Automated decision-making system for customer support.

### 2. Human AI Performance Visualizer (Creative)
An innovative visualization project that compares four different ML models.
- **Models Compared:** KNN, SVM, Decision Trees, and Random Forest.
- **Creative Element:** Performance metrics (Precision, Accuracy, Recall, F1) are dynamically mapped to a human character's body parts and colors.
- **Outcome:** A deep understanding of model comparison and data storytelling.

## 🛠️ Tech Stack:
- **Python**, **Scikit-learn**, **Matplotlib**, **Pandas**, **Imbalanced-learn**.
